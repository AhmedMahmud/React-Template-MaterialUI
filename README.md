# React-Template-MaterialUI
This repository is a React Template which includes React, Babel, Webpackand MaterialUI.
It is an upgarde from my simple React-Template repository, this one includes [Material-UI](http://www.material-ui.com/) 
which is a framework for React Components which styles off of [Google's Material Design](https://material.google.com/).

## Installation
After forking and cloning this repository
Run 
```sh
$ npm install
```

After everything is installed, you are ready to roll!

## Usage
When you are ready to see your application, Simply run
```sh
$ npm start
```
Which will fire up Webpack!

You can now see your application at [localhost:8080](http://localhost:8080)
